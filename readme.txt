== Installation ==

This section describes how to install the plugin and get it working.

1. Upload whole folder to the `/wp-content/plugins/` directory
2. Get your client id from [https://dna.daum.net/myapi/affiliate/new?mode=login](https://dna.daum.net/myapi/affiliate/new?mode=login)
3. Put your $clientid in daum_comment.php
4. Activate the plugin through the 'Plugins' menu in WordPress

Daum (Korean: 다음; KRX: 035720) is a web portal in South Korea, like Naver and Nate. Daum offers many Internet services to web users, including a popular free web-based e-mail, messaging service, forums, shopping and news.
